kablammo
========

Kablammo!  the GoGaRuCo death match


Setup
-----

- Clone the repo
- Install dependancies: run `bundle`
- Start the server: run `rerun.sh`
- Run two or more strategy clients (see below)
- Create a new board `localhost:4567/boards/new`
